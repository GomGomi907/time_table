Time Table Frontend Draft

주간 운영 화면을 위한 HTML/CSS 시안 3종

일정, 할 일, 목표, 집중 실행 화면을 각각 다른 강조점으로 풀어낸 정적 목업입니다. 빠르게 비교하고 다음 단계에서 Next.js 화면 구조로 옮기기 쉽게 만들었습니다.

Dashboard Schedule Focus Responsive
01 Overview

오늘 브리핑

오늘 예정된 일정과 바로 확인할 조정안만 먼저 보여주는 홈 화면입니다.

  • 오늘 일정 먼저 확인
  • 조정안 승인/나중에 검토
  • 필요 정보만 요약
02 Planner

주간 플래너

5분 단위 시간표 경험을 염두에 두고, 주간 그리드와 미배치 할 일 레일을 함께 배치한 화면입니다.

  • 주간 보드 중심
  • 오른쪽 실행 레일
  • 충돌/우선순위 시각화
03 Execution

집중 실행 화면

단일 작업 원칙에 맞춘 실행 전용 화면입니다. 지금 할 일, 남은 시간, 다음 일정을 크게 보여줍니다.

  • 현재 작업 1개 강조
  • 다음 일정 안내
  • 미루기/연장 액션 명확화
